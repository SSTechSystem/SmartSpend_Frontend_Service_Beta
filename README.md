# sstechkit_front

## Frontend setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### To test local build
```
npm run preview
```

### To update the package
```
mention the latest stable version of that package in package.json file.
Run command
npm install
```


## Backend Technologies Used
- Backend: Node.js
- DB: MySQL
